# Component Documentation

The Elektrik âš¡ Design System provides a collection of React components designed for consistency, accessibility, and ease of use. This documentation explains how to use each component in your projects.

## Getting Started

To use Elektrik components in your React application:

```jsx
import { Button, Input, Alert } from 'elektrik-design-system';
import { GlobalStyles } from 'elektrik-design-system';

function App() {
  return (
    <>
      <GlobalStyles /> {/* Apply global styles */}
      <div className="app">
        <Button variant="primary">Click Me</Button>
        <Input label="Username" placeholder="Enter username" />
        <Alert variant="success">Operation completed successfully!</Alert>
      </div>
    </>
  );
}
```

## Design Tokens

Elektrik uses a consistent set of design tokens for colors, typography, spacing, and more. You can import and use these tokens in your custom components:

```jsx
import { colors, typography, spacing, borderRadius } from 'elektrik-design-system';

// Use in styled-components
const CustomElement = styled.div`
  color: ${colors.primary};
  font-family: ${typography.fontFamily};
  padding: ${spacing.md};
  border-radius: ${borderRadius.md};
`;
```

## Components

Each component in the Elektrik Design System is designed to be:

- **Accessible**: Following WCAG guidelines
- **Customizable**: Through props and theming
- **Responsive**: Works across various screen sizes
- **Consistent**: Following the same design patterns

Navigate through the documentation sections to learn about each component, its props, and usage examples.
